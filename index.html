🎧 Automatic Narration for Vinh Khanh Food Street

An automatic narration system developed for Vinh Khanh Food Street, designed to enhance the visitor experience by providing interactive audio introductions for famous local dishes and food stalls.

This project was developed as part of the Software Engineering Seminar course, focusing on the Vibe Coding approach, where rapid prototyping and creative interaction design are emphasized.

View on GitHub
  • Frontend: Built with React.js, providing a responsive and interactive interface that allows users to explore food stalls and trigger automatic narration.
  • Backend: Implemented with Spring Boot, delivering RESTful APIs for managing food stall information, narration content, and user interactions.
  • Core Idea: When users browse or select a location or dish, the system automatically generates or plays narration describing the food, its origin, and its cultural background.
  • Purpose: The system aims to support digital tourism and cultural storytelling, helping visitors learn about the culinary culture of Vinh Khanh Food Street in an engaging way.
  • Architecture: Designed using a decoupled frontend-backend architecture, enabling scalable development and easy integration with future services such as text-to-speech or AI narration.
