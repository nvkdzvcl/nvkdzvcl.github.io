Local-First Retrieval-Augmented Generation (RAG) System

Designed and implemented a fully local Retrieval-Augmented Generation (RAG) pipeline for private document question-answering. The system supports semantic retrieval over PDF and DOCX knowledge bases while preserving data privacy through on-device inference with open-source large language models.

View on GitHub
  • RAG Pipeline: Implemented document ingestion, recursive text chunking, embedding generation, semantic retrieval, and context-aware answer generation.
  • Framework & Orchestration: Built modular retrieval workflows using LangChain for chaining retrievers, prompts, and local LLM inference.
  • Vector Search: Integrated ChromaDB for efficient vector storage, similarity search, and persistent knowledge indexing.
  • Local LLM Inference: Deployed Llama 3 models through Ollama to enable fully offline, low-latency inference without external APIs.
  • Privacy & Performance: Optimized the system for local-first execution, minimizing cloud dependency while maintaining responsive retrieval quality.

AI-Powered Automatic Narration System for Vinh Khanh Food Street

Developed an interactive automatic narration platform for Vinh Khanh Food Street, aimed at supporting digital tourism and cultural storytelling through AI-assisted audio experiences. The system enables visitors to explore local dishes, food stalls, and culinary history through dynamically generated narration content.

This project was built as part of a Software Engineering Seminar, with a strong focus on rapid prototyping, user-centered interaction design, and modern full-stack development practices.

View on GitHub
  • Frontend Development: Built a responsive user interface using React.js, allowing users to browse food stalls, view culinary information, and trigger narration interactions.
  • Backend Services: Developed RESTful APIs with Spring Boot for managing food data, narration content, and communication between frontend and backend services.
  • Narration Workflow: Designed a narration pipeline capable of presenting descriptive audio content about dishes, cultural origins, and local culinary stories based on user interactions.
  • System Architecture: Applied a decoupled frontend-backend architecture to improve scalability, maintainability, and future AI/TTS integration capabilities.
  • Project Objective: Focused on enhancing visitor engagement through interactive digital experiences and preserving local culinary culture using modern software technologies.
